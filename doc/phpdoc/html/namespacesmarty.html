This command line script rips gettext strings from smarty file, and prints them to stdout in C format, that can later be used with the standard gettext tools.

Usage: ./tsmarty2c.php <filename or="" directory>=""> <file2> <..> > smarty.c

If a parameter is a directory, the template files within will be parsed.

-gettext
